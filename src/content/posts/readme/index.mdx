The following files are compatible with this document:

- `src/content/site.json` - homepage and site content source of truth.
- `src/lib/site-schema.ts` - validates the `site.json` structure.
- `src/lib/site-data.ts` - loads the `default` site entry.
- `src/pages/index.astro` - homepage template.
- `src/components/base/HeroMetric.astro` - hero metric display component.
- `src/components/base/SocialLink.astro` - social icon link component.
- `src/config.ts` - theme behavior and posts config.

### Site Content

Homepage content now lives in `src/content/site.json`, not in `src/config.ts`.

```json title="site.json"
{
  "default": {
    "site": {
      "title": "Litos",
      "description": "Your site description",
      "website": "https://example.com",
      "lang": "en",
      "author": "Your name",
      "ogImage": "/og-image.webp"
    }
  }
}
```

The `site` object controls the shared metadata used in layout, head tags, feeds, and the homepage.

### Navigation Links

Header and footer navigation also come from `src/content/site.json`:

```json title="site.json"
{
  "headerLinks": [{ "name": "Posts", "url": "/posts" }],
  "footerLinks": [
    { "name": "Readme", "url": "/" },
    { "name": "Posts", "url": "/posts" },
    { "name": "Tags", "url": "/tags" }
  ]
}
```

### Hero Content

The homepage hero reads from `hero`, `heroMetric`, and `nap`.

```json title="site.json"
{
  "hero": {
    "brand": "Litos",
    "tagline": "A Simple & Modern Blog Theme for Astro",
    "pitch": "Short homepage description"
  },
  "heroMetric": {
    "icon": "icon-[mdi--star]",
    "primaryValue": "4.9",
    "primaryLabel": "/",
    "secondaryValue": 50,
    "secondaryLabel": "reviews"
  },
  "nap": {
    "company": "Northwind Dental Studio",
    "status": "Open now",
    "address": "1284 Market Street, Suite 300, Portland, OR 97205",
    "phone": "(503) 555-0147"
  }
}
```

The hero description is truncated to 150 characters by `truncateText()` before rendering.

### Social Links

Social icons below the hero also come from `src/content/site.json`. Icons are from :link[Iconify]{id=https://icon-sets.iconify.design/}.

```json title="site.json"
{
  "socialLinks": [
    {
      "name": "facebook",
      "url": "https://facebook.com/yourpage",
      "icon": "icon-[ri--facebook-line]"
    }
  ]
}
```

Optional fields:

- `count` - an optional number shown beside the icon.
- `label` - an optional text label shown after `count`.

### Home Sections

Additional homepage sections are configured in `homeSections` and rendered above the posts list.

```json title="site.json"
{
  "homeSections": [
    {
      "key": "location",
      "title": "Location",
      "description": "A dedicated section for location details will be added here."
    }
  ]
}
```

### Posts

The homepage still renders pinned posts first. If there are no pinned posts, it renders the latest posts using `POSTS_CONFIG.homePageConfig`.
